How to Run

Requires JDK 17 or later.

# Option 1: Use the shell script
./run.sh

# Option 2: Compile and run manually
mkdir -p out
javac -d out src/mario/util/*.java src/mario/entities/*.java src/mario/world/*.java src/mario/*.java
java -cp out mario.Main
